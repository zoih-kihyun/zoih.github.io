# 오름 솔루션

## 주요 기능

### 장비 연결

오름은 진단검사의학과의 검사실 및 검사 환경 전반에서 운용되는 160종 이상의 다양한 의료장비와 인터페이스합니다. 검체검사 자동화(TLA), 임상화학, 면역혈청, 진단혈액, 요검사, 미생물, 분자진단, 혈액은행 등 검사실 장비는 물론, 병동, 응급실 등에서 운용되는 현장검사(POCT) 장비까지 통합 지원합니다.

### 데이터 관리

오름은 검사 프로그램의 모든 측면을 관리하는 메뉴 기반 시스템을 제공합니다. 장비로부터 수신된 모든 환자 결과 및 정도관리(QC) 결과 조회, 검사자 정보 가져오기 및 관리, 장비 설정, 인터페이스 예외 처리, 리포팅 등 다양한 기능을 포함합니다.

## 지원 장비 분류

| 분류 | 설명 |
|------|------|
| **검체검사 자동화(TLA)** | 검체 접수부터 원심분리, 분주, 이송, 검사 후 보관까지 전 과정을 연결하는 통합 시스템 |
| **임상화학 (Chemistry)** | 간 기능, 신장 기능, 당뇨, 지질 등 측정 (가장 데이터량이 많음) |
| **면역혈청 (Immunoassay)** | 항원-항체 반응을 이용한 암 표지자, 호르몬, 감염성 질환 검사 |
| **진단혈액 (Hematology)** | CBC(일반혈액검사), 혈액 응고 검사, 망상적혈구 측정 등 |
| **요검사 (Urinalysis)** | 요시험지봉 분석기, 요침사 분석기 |
| **미생물 (Microbiology)** | 미생물 배양, 동정(ID), 항생제 감수성 검사(AST) |
| **분자진단 (Molecular)** | PCR, 시퀀싱(NGS) 등 유전 정보 분석 |
| **혈액은행 (Blood Bank)** | 혈액형 판정 및 교차시험 자동화 |
| **현장검사 (POCT)** | 혈당 측정기(BST), 혈액 가스 분석기(ABGA) 등 병동·응급실 소형 장비 |
| **특수검사** | 유세포 분석기(Flow Cytometry), 질량 분석기(LC-MS/MS) 등 |
